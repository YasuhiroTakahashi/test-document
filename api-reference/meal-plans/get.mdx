## 概要

このエンドポイントは、献立IDを指定して、その献立の完全な詳細情報を取得します。すべての食事、レシピ、材料、栄養情報が含まれます。

## エンドポイント

```
GET https://api.mealplan-ai.com/v1/meal-plans/{id}
```

## 認証

<ParamField header="Authorization" type="string" required>
  Bearer トークン形式のAPIキーまたはOAuthアクセストークン
</ParamField>

## パスパラメータ

<ParamField path="id" type="string" required>
  献立のユニークID
  
  例: `mp_abc123`
</ParamField>

## リクエスト例

<CodeGroup>
```bash cURL
curl -X GET https://api.mealplan-ai.com/v1/meal-plans/mp_abc123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://api.mealplan-ai.com/v1/meal-plans/mp_abc123',
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);
const mealPlan = await response.json();
console.log(mealPlan);
```

```python Python
import requests

meal_plan_id = "mp_abc123"
url = f"https://api.mealplan-ai.com/v1/meal-plans/{meal_plan_id}"
headers = {
    "Authorization": "Bearer YOUR_API_KEY"
}

response = requests.get(url, headers=headers)
meal_plan = response.json()
print(meal_plan)
```

```ruby Ruby
require 'net/http'
require 'json'

meal_plan_id = 'mp_abc123'
uri = URI("https://api.mealplan-ai.com/v1/meal-plans/#{meal_plan_id}")
request = Net::HTTP::Get.new(uri)
request['Authorization'] = 'Bearer YOUR_API_KEY'

response = Net::HTTP.start(uri.hostname, uri.port, use_ssl: true) do |http|
  http.request(request)
end

meal_plan = JSON.parse(response.body)
puts meal_plan
```
</CodeGroup>

## 使用例

### 献立の詳細を取得して表示

```javascript
async function displayMealPlan(mealPlanId) {
  const response = await fetch(
    `https://api.mealplan-ai.com/v1/meal-plans/${mealPlanId}`,
    {
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY'
      }
    }
  );
  
  const { data } = await response.json();
  
  console.log(`献立期間: ${data.start_date} 〜 ${data.end_date}`);
  console.log(`平均カロリー: ${data.nutrition_summary.avg_daily_calories} kcal/日`);
  
  data.meals.forEach(meal => {
    console.log(`\n${meal.date}:`);
    console.log(`  朝食: ${meal.breakfast.name}`);
    console.log(`  昼食: ${meal.lunch.name}`);
    console.log(`  夕食: ${meal.dinner.name}`);
  });
}

displayMealPlan('mp_abc123');
```

### レシピの材料を抽出

```javascript
async function getIngredients(mealPlanId) {
  const response = await fetch(
    `https://api.mealplan-ai.com/v1/meal-plans/${mealPlanId}`,
    {
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY'
      }
    }
  );
  
  const { data } = await response.json();
  const allIngredients = [];
  
  data.meals.forEach(meal => {
    ['breakfast', 'lunch', 'dinner'].forEach(mealType => {
      if (meal[mealType]) {
        allIngredients.push(...meal[mealType].ingredients);
      }
    });
  });
  
  return allIngredients;
}
```
