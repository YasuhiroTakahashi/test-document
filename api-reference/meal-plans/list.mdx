## 概要

このエンドポイントは、アカウントに紐づく献立の一覧を取得します。ページネーション、フィルタリング、ソートをサポートしています。

## エンドポイント

```
GET https://api.mealplan-ai.com/v1/meal-plans
```

## 認証

<ParamField header="Authorization" type="string" required>
  Bearer トークン形式のAPIキーまたはOAuthアクセストークン
</ParamField>

## クエリパラメータ

<ParamField query="page" type="integer" default="1">
  ページ番号
</ParamField>

<ParamField query="per_page" type="integer" default="20">
  1ページあたりの件数（最大: 100）
</ParamField>

<ParamField query="start_date" type="string">
  開始日でフィルタリング（この日付以降の献立）
  
  例: `2024-01-01`
</ParamField>

<ParamField query="end_date" type="string">
  終了日でフィルタリング（この日付以前の献立）
  
  例: `2024-12-31`
</ParamField>

<ParamField query="profile_id" type="string">
  特定のプロフィールIDでフィルタリング
</ParamField>

<ParamField query="sort" type="string" default="created_at">
  ソート項目
  
  選択肢: `created_at`, `start_date`, `duration_days`
</ParamField>

<ParamField query="order" type="string" default="desc">
  ソート順
  
  選択肢: `asc`, `desc`
</ParamField>

## リクエスト例

<CodeGroup>
```bash cURL
curl -X GET "https://api.mealplan-ai.com/v1/meal-plans?page=1&per_page=20" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://api.mealplan-ai.com/v1/meal-plans?page=1&per_page=20',
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);
const data = await response.json();
console.log(data);
```

```python Python
import requests

url = "https://api.mealplan-ai.com/v1/meal-plans"
params = {
    "page": 1,
    "per_page": 20
}
headers = {
    "Authorization": "Bearer YOUR_API_KEY"
}

response = requests.get(url, params=params, headers=headers)
data = response.json()
print(data)
```
</CodeGroup>

## フィルタリング例

### 日付範囲でフィルタリング

```bash
curl -X GET "https://api.mealplan-ai.com/v1/meal-plans?start_date=2024-01-01&end_date=2024-01-31" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### プロフィールでフィルタリング

```bash
curl -X GET "https://api.mealplan-ai.com/v1/meal-plans?profile_id=prof_abc123" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### ソート順を変更

```bash
curl -X GET "https://api.mealplan-ai.com/v1/meal-plans?sort=start_date&order=asc" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
