## 概要

このエンドポイントは、レシピIDを指定して、材料、調理手順、栄養情報などの詳細を取得します。

## エンドポイント

```
GET https://api.mealplan-ai.com/v1/recipes/{id}
```

## 認証

<ParamField header="Authorization" type="string" required>
  Bearer トークン形式のAPIキーまたはOAuthアクセストークン
</ParamField>

## パスパラメータ

<ParamField path="id" type="string" required>
  レシピのユニークID
  
  例: `rec_001`
</ParamField>

## リクエスト例

<CodeGroup>
```bash cURL
curl -X GET https://api.mealplan-ai.com/v1/recipes/rec_001 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://api.mealplan-ai.com/v1/recipes/rec_001',
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);

const recipe = await response.json();
console.log(recipe);
```

```python Python
import requests

recipe_id = "rec_001"
url = f"https://api.mealplan-ai.com/v1/recipes/{recipe_id}"
headers = {
    "Authorization": "Bearer YOUR_API_KEY"
}

response = requests.get(url, headers=headers)
recipe = response.json()
print(recipe)
```
</CodeGroup>

## 使用例

### レシピを表示

```javascript
async function displayRecipe(recipeId) {
  const response = await fetch(
    `https://api.mealplan-ai.com/v1/recipes/${recipeId}`,
    {
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY'
      }
    }
  );
  
  const { data: recipe } = await response.json();
  
  console.log(`料理名: ${recipe.name}`);
  console.log(`調理時間: ${recipe.cooking_time}分`);
  console.log(`カロリー: ${recipe.calories} kcal`);
  
  console.log('\n材料:');
  recipe.ingredients.forEach(ing => {
    console.log(`- ${ing.name}: ${ing.amount}${ing.unit}`);
  });
  
  console.log('\n作り方:');
  recipe.instructions.forEach(inst => {
    console.log(`${inst.step}. ${inst.description}`);
  });
}

displayRecipe('rec_001');
```
