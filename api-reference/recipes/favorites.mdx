## 概要

このエンドポイントは、レシピをお気に入りに追加したり、削除したりできます。お気に入りレシピは献立生成時に優先的に使用されます。

## お気に入りに追加

### エンドポイント

```
POST https://api.mealplan-ai.com/v1/recipes/{id}/favorite
```

### パスパラメータ

<ParamField path="id" type="string" required>
  レシピID
</ParamField>

### リクエスト例

<CodeGroup>
```bash cURL
curl -X POST https://api.mealplan-ai.com/v1/recipes/rec_001/favorite \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://api.mealplan-ai.com/v1/recipes/rec_001/favorite',
  {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);

const result = await response.json();
```

```python Python
import requests

recipe_id = "rec_001"
url = f"https://api.mealplan-ai.com/v1/recipes/{recipe_id}/favorite"
headers = {
    "Authorization": "Bearer YOUR_API_KEY"
}

response = requests.post(url, headers=headers)
result = response.json()
```
</CodeGroup>

### レスポンス

```json
{
  "success": true,
  "data": {
    "recipe_id": "rec_001",
    "is_favorite": true,
    "favorited_at": "2024-01-15T10:30:00Z"
  }
}
```

## お気に入りから削除

### エンドポイント

```
DELETE https://api.mealplan-ai.com/v1/recipes/{id}/favorite
```

### リクエスト例

<CodeGroup>
```bash cURL
curl -X DELETE https://api.mealplan-ai.com/v1/recipes/rec_001/favorite \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://api.mealplan-ai.com/v1/recipes/rec_001/favorite',
  {
    method: 'DELETE',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);
```
</CodeGroup>

### レスポンス

```json
{
  "success": true,
  "data": {
    "recipe_id": "rec_001",
    "is_favorite": false
  }

```

## お気に入り一覧取得

### エンドポイント

```
GET https://api.mealplan-ai.com/v1/recipes/favorites
```

### クエリパラメータ

<ParamField query="page" type="integer" default="1">
  ページ番号
</ParamField>

<ParamField query="per_page" type="integer" default="20">
  1ページあたりの件数
</ParamField>

### リクエスト例

curl -X GET https://api.mealplan-ai.com/v1/recipes/favorites \

## 使用例

### お気に入りトグル機能

```javascript
async function toggleFavorite(recipeId, isFavorite) {
  const method = isFavorite ? 'DELETE' : 'POST';
  const response = await fetch(
    `https://api.mealplan-ai.com/v1/recipes/${recipeId}/favorite`,
    {
      method: method,
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY'
      }
    }
  );
  
  return await response.json();
}
```
