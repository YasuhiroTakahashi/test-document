## 概要

このエンドポイントは、様々な条件でレシピを検索できます。料理名、材料、調理時間、難易度などで絞り込みが可能です。

## エンドポイント

```
GET https://api.mealplan-ai.com/v1/recipes
```

## 認証

<ParamField header="Authorization" type="string" required>
  Bearer トークン形式のAPIキーまたはOAuthアクセストークン
</ParamField>

## クエリパラメータ

<ParamField query="q" type="string">
  検索キーワード（料理名、材料名など）
  
  例: `鶏肉`
</ParamField>

<ParamField query="cuisine_type" type="string">
  料理のジャンル
  
  選択肢: `japanese`, `western`, `chinese`, `italian`, `french`, `korean`, `thai`, `indian`, `mexican`
</ParamField>

<ParamField query="max_cooking_time" type="integer">
  最大調理時間（分）
  
  例: `30`
</ParamField>

<ParamField query="difficulty" type="string">
  難易度
  
  選択肢: `easy`, `medium`, `hard`
</ParamField>

<ParamField query="max_calories" type="integer">
  最大カロリー（kcal）
  
  例: `500`
</ParamField>

<ParamField query="ingredients" type="string">
  含む材料（カンマ区切り）
  
  例: `鶏肉,玉ねぎ`
</ParamField>

<ParamField query="exclude_ingredients" type="string">
  除外する材料（カンマ区切り）
  
  例: `えび,かに`
</ParamField>

<ParamField query="tags" type="string">
  タグでフィルタリング（カンマ区切り）
  
  例: `時短,ヘルシー`
</ParamField>

<ParamField query="page" type="integer" default="1">
  ページ番号
</ParamField>

<ParamField query="per_page" type="integer" default="20">
  1ページあたりの件数（最大: 100）
</ParamField>

<ParamField query="sort" type="string" default="relevance">
  ソート項目
  
  選択肢: `relevance`, `cooking_time`, `calories`, `popularity`
</ParamField>

## リクエスト例

<CodeGroup>
```bash cURL
curl -X GET "https://api.mealplan-ai.com/v1/recipes?q=鶏肉&max_cooking_time=30&difficulty=easy" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const params = new URLSearchParams({
  q: '鶏肉',
  max_cooking_time: '30',
  difficulty: 'easy'
});

const response = await fetch(
  `https://api.mealplan-ai.com/v1/recipes?${params}`,
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);

const recipes = await response.json();
console.log(recipes);
```

```python Python
import requests

url = "https://api.mealplan-ai.com/v1/recipes"
params = {
    "q": "鶏肉",
    "max_cooking_time": 30,
    "difficulty": "easy"
}
headers = {
    "Authorization": "Bearer YOUR_API_KEY"
}

response = requests.get(url, params=params, headers=headers)
recipes = response.json()
print(recipes)
```
</CodeGroup>

## 検索例

### 時短レシピを検索

```bash
curl -X GET "https://api.mealplan-ai.com/v1/recipes?max_cooking_time=20&tags=時短" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### ヘルシーな和食を検索

```bash
curl -X GET "https://api.mealplan-ai.com/v1/recipes?cuisine_type=japanese&max_calories=400&tags=ヘルシー" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### 特定の材料を使ったレシピ

```bash
curl -X GET "https://api.mealplan-ai.com/v1/recipes?ingredients=鶏肉,玉ねぎ&exclude_ingredients=えび" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## 使用例

### レシピ検索フォーム

```javascript
async function searchRecipes(filters) {
  const params = new URLSearchParams();
  
  if (filters.keyword) params.append('q', filters.keyword);
  if (filters.cuisineType) params.append('cuisine_type', filters.cuisineType);
  if (filters.maxTime) params.append('max_cooking_time', filters.maxTime);
  if (filters.difficulty) params.append('difficulty', filters.difficulty);
  
  const response = await fetch(
    `https://api.mealplan-ai.com/v1/recipes?${params}`,
    {
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY'
      }
    }
  );
  
  return await response.json();
}

// 使用例
const results = await searchRecipes({
  keyword: '鶏肉',
  maxTime: 30,
  difficulty: 'easy'
});
```
