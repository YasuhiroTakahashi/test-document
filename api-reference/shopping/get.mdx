## 概要

買い物リストIDを指定して、詳細情報を取得します。

## エンドポイント

```
GET https://api.mealplan-ai.com/v1/shopping-lists/{id}
```

## 認証

<ParamField header="Authorization" type="string" required>
  Bearer トークン形式のAPIキーまたはOAuthアクセストークン
</ParamField>

## パスパラメータ

<ParamField path="id" type="string" required>
  買い物リストID
</ParamField>

## リクエスト例

<CodeGroup>
```bash cURL
curl -X GET https://api.mealplan-ai.com/v1/shopping-lists/sl_xyz789 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://api.mealplan-ai.com/v1/shopping-lists/sl_xyz789',
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);

const shoppingList = await response.json();
```
</CodeGroup>
