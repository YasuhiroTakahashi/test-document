description: "買い物リストのアイテムを更新します"
---

## 概要

買い物リストのアイテムのチェック状態や数量を更新します。

## エンドポイント

```
PATCH https://api.mealplan-ai.com/v1/shopping-lists/{id}/items/{item_id}
```

## 認証

<ParamField header="Authorization" type="string" required>
  Bearer トークン形式のAPIキーまたはOAuthアクセストークン
</ParamField>

## パスパラメータ

<ParamField path="id" type="string" required>
  買い物リストID
</ParamField>

<ParamField path="item_id" type="string" required>
  アイテムID
</ParamField>

## リクエストボディ

<ParamField body="checked" type="boolean">
  チェック状態
</ParamField>

<ParamField body="amount" type="string">
  数量
</ParamField>

## リクエスト例

<CodeGroup>
```bash cURL
curl -X PATCH https://api.mealplan-ai.com/v1/shopping-lists/sl_xyz789/items/item_001 \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"checked": true}'
```

```javascript JavaScript
const response = await fetch(
  'https://api.mealplan-ai.com/v1/shopping-lists/sl_xyz789/items/item_001',
  {
    method: 'PATCH',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      checked: true
    })
  }
);

const result = await response.json();
```

```python Python
import requests

url = "https://api.mealplan-ai.com/v1/shopping-lists/sl_xyz789/items/item_001"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "checked": True
}

response = requests.patch(url, headers=headers, json=data)
result = response.json()
```
</CodeGroup>

## 使用例

### アイテムをチェック

```javascript
async function checkItem(listId, itemId) {
  const response = await fetch(
    `https://api.mealplan-ai.com/v1/shopping-lists/${listId}/items/${itemId}`,
    {
      method: 'PATCH',
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({ checked: true })
    }
  );
  
  return await response.json();
}
```
