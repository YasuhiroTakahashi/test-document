## 概要

登録されている家族プロフィールの情報を取得します。

## エンドポイント

```
GET https://api.mealplan-ai.com/v1/profiles
```

## 認証

<ParamField header="Authorization" type="string" required>
  Bearer トークン形式のAPIキーまたはOAuthアクセストークン
</ParamField>

## リクエスト例

<CodeGroup>
```bash cURL
curl -X GET https://api.mealplan-ai.com/v1/profiles \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://api.mealplan-ai.com/v1/profiles',
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);

const profiles = await response.json();
```

```python Python
import requests

url = "https://api.mealplan-ai.com/v1/profiles"
headers = {
    "Authorization": "Bearer YOUR_API_KEY"
}

response = requests.get(url, headers=headers)
profiles = response.json()
```
</CodeGroup>

## レスポンス

<ResponseField name="success" type="boolean">
  リクエストが成功したかどうか
</ResponseField>

<ResponseField name="data" type="array">
  プロフィールデータの配列
  
  <Expandable title="data の要素">
    <ResponseField name="data[].id" type="string">
      プロフィールID
    </ResponseField>
    
    <ResponseField name="data[].name" type="string">
      名前
    </ResponseField>
    
    <ResponseField name="data[].age" type="integer">
      年齢
    </ResponseField>
    
    <ResponseField name="data[].gender" type="string">
      性別（male/female/other）
    </ResponseField>
    
    <ResponseField name="data[].allergies" type="array">
      アレルギー情報
    </ResponseField>
    
    <ResponseField name="data[].dislikes" type="array">
      苦手な食材
    </ResponseField>
    
    <ResponseField name="data[].dietary_restrictions" type="array">
      食事制限（vegetarian, vegan, halal等）
    </ResponseField>
    
    <ResponseField name="data[].health_goals" type="object">
      健康目標
    </ResponseField>
  </Expandable>
</ResponseField>
