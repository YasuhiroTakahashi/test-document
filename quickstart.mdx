## 3ステップで始める

MealPlan AIを使って、最初の献立を生成するまでの簡単なガイドです。

<Steps>
  <Step title="アカウント作成とAPIキー取得">
    まず、MealPlan AIのアカウントを作成し、APIキーを取得します。
    
    1. [ダッシュボード](https://app.mealplan-ai.com/signup)にアクセス
    2. メールアドレスとパスワードでアカウント作成
    3. ダッシュボードの「API Keys」セクションから新しいAPIキーを生成
    
    <Note>
      APIキーは安全に保管してください。他人と共有しないでください。
    </Note>
  </Step>

  <Step title="家族プロフィールの設定">
    家族の情報を登録して、最適な献立を生成できるようにします。
    
    <Tabs>
      <Tab title="Webダッシュボード">
        1. ダッシュボードの「Family Profile」に移動
        2. 家族メンバーを追加（名前、年齢、性別）
        3. アレルギー情報や食材の好き嫌いを設定
        4. 「保存」をクリック
      </Tab>
      
      <Tab title="API">
        APIを使用してプロフィールを作成することもできます。
        
        ```bash
        curl -X POST https://api.mealplan-ai.com/v1/profiles \
          -H "Authorization: Bearer YOUR_API_KEY" \
          -H "Content-Type: application/json" \
          -d '{
            "family_members": [
              {
                "name": "太郎",
                "age": 35,
                "gender": "male",
                "allergies": ["えび", "かに"]
              },
              {
                "name": "花子",
                "age": 33,
                "gender": "female",
                "dietary_restrictions": ["vegetarian"]
              },
              {
                "name": "一郎",
                "age": 8,
                "gender": "male",
                "dislikes": ["ピーマン", "にんじん"]
              }
            ]
          }'
        ```
      </Tab>
    </Tabs>
    
    <Tip>
      詳細な設定方法は[家族プロフィールガイド](/user-guide/family-profile)をご覧ください。
    </Tip>
  </Step>

  <Step title="最初の献立を生成">
    いよいよ献立を生成してみましょう！
    
    <Tabs>
      <Tab title="Webダッシュボード">
        1. ダッシュボードの「Meal Plans」に移動
        2. 「新しい献立を生成」ボタンをクリック
        3. 期間（1週間、3日間など）を選択
        4. 「生成」をクリック
        5. 数秒で献立が表示されます！
      </Tab>
      
      <Tab title="cURL">
        ```bash
        curl -X POST https://api.mealplan-ai.com/v1/meal-plans/generate \
          -H "Authorization: Bearer YOUR_API_KEY" \
          -H "Content-Type: application/json" \
          -d '{
            "duration_days": 7,
            "preferences": {
              "cuisine_types": ["japanese", "western"],
              "max_cooking_time": 45
            }
          }'
        ```
      </Tab>
      
      <Tab title="JavaScript">
        ```javascript
        const response = await fetch('https://api.mealplan-ai.com/v1/meal-plans/generate', {
          method: 'POST',
          headers: {
            'Authorization': 'Bearer YOUR_API_KEY',
            'Content-Type': 'application/json'
          },
          body: JSON.stringify({
            duration_days: 7,
            preferences: {
              cuisine_types: ['japanese', 'western'],
              max_cooking_time: 45
            }
          })
        });
        
        const mealPlan = await response.json();
        console.log('献立が生成されました:', mealPlan);
        ```
      </Tab>
      
      <Tab title="Python">
        ```python
        import requests
        
        url = "https://api.mealplan-ai.com/v1/meal-plans/generate"
        headers = {
            "Authorization": "Bearer YOUR_API_KEY",
            "Content-Type": "application/json"
        }
        data = {
            "duration_days": 7,
            "preferences": {
                "cuisine_types": ["japanese", "western"],
                "max_cooking_time": 45
            }
        }
        
        response = requests.post(url, headers=headers, json=data)
        meal_plan = response.json()
        print("献立が生成されました:", meal_plan)
        ```
      </Tab>
    </Tabs>
    
    <Check>
      おめでとうございます！最初の献立が生成されました。
    </Check>
  </Step>
</Steps>

## よくある質問

<AccordionGroup>
  <Accordion title="無料プランでどこまで使えますか？">
    無料プランでは、月に10回まで献立を生成できます。また、基本的な栄養分析機能も利用可能です。より多くの献立生成や高度な機能が必要な場合は、[料金プラン](/pricing)をご確認ください。
  </Accordion>
  
  <Accordion title="APIキーはどこで確認できますか？">
    [ダッシュボード](https://app.mealplan-ai.com/settings/api-keys)の「Settings」→「API Keys」セクションで確認・管理できます。
  </Accordion>
  
  <Accordion title="献立の生成にどのくらい時間がかかりますか？">
    通常、3〜5秒程度で献立が生成されます。家族構成や条件が複雑な場合は、最大10秒程度かかることがあります。
  </Accordion>
  
  <Accordion title="生成された献立を編集できますか？">
    はい、生成後に個別のレシピを入れ替えたり、食材を調整したりできます。詳しくは[献立管理ガイド](/user-guide/meal-planning)をご覧ください。
  </Accordion>
  
  <Accordion title="アレルギー情報は必須ですか？">
    必須ではありませんが、設定することで安全な献立を生成できます。特に重度のアレルギーがある場合は、必ず設定することをお勧めします。
  </Accordion>
</AccordionGroup>

## トラブルシューティング

<Warning>
  **認証エラーが発生する場合**
  
  - APIキーが正しいか確認してください
  - `Authorization: Bearer YOUR_API_KEY`の形式で送信しているか確認
  - APIキーの有効期限が切れていないか確認
</Warning>

<Warning>
  **献立生成に失敗する場合**
  
  - 家族プロフィールが正しく設定されているか確認
  - リクエストのJSON形式が正しいか確認
  - レート制限に達していないか確認（[レート制限ガイド](/developer/rate-limits)参照）
</Warning>
