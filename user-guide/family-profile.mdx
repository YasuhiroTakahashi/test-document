## 概要

家族プロフィールを設定することで、MealPlan AIは各家族メンバーの年齢、アレルギー、食の好みを考慮した最適な献立を提案できます。

## 家族プロフィールの作成

<Steps>
  <Step title="ダッシュボードにアクセス">
    [ダッシュボード](https://app.mealplan-ai.com)にログインし、「Family Profile」セクションに移動します。
  </Step>
  
  <Step title="家族メンバーを追加">
    「Add Family Member」ボタンをクリックして、家族メンバーの情報を入力します。
    
    **必須項目:**
    - 名前
    - 年齢
    - 性別
    
    **任意項目:**
    - アレルギー情報
    - 苦手な食材
    - 食事制限（ベジタリアン、ハラルなど）
    - 健康目標
  </Step>
  
  <Step title="情報を保存">
    すべての情報を入力したら、「Save」ボタンをクリックして保存します。
  </Step>
</Steps>

## 設定項目の詳細

### 基本情報

<AccordionGroup>
  <Accordion title="名前">
    家族メンバーの名前を入力します。献立表示時に使用されます。
  </Accordion>
  
  <Accordion title="年齢">
    年齢に応じて、適切なカロリーや栄養素の量が自動調整されます。
    
    - 子供（3-12歳）: 成長に必要な栄養素を重視
    - 青年（13-19歳）: 高いエネルギー需要に対応
    - 成人（20-64歳）: バランスの取れた栄養
    - 高齢者（65歳以上）: 消化しやすい食材を優先
  </Accordion>
  
  <Accordion title="性別">
    性別により、推奨カロリーや栄養素の量が異なります。
  </Accordion>
</AccordionGroup>

### アレルギー情報

<Warning>
  **重要**: アレルギー情報は正確に入力してください。この情報に基づいて、該当する食材を含むレシピは自動的に除外されます。
</Warning>

**一般的なアレルギー食材:**
- えび、かに
- 小麦
- そば
- 卵
- 乳製品
- ピーナッツ
- 大豆

複数のアレルギーがある場合は、すべて登録してください。

### 苦手な食材

アレルギーではないが、苦手な食材を登録できます。献立生成時に、これらの食材を避けたレシピが提案されます。

**例:**
- ピーマン
- セロリ
- パクチー
- レバー

<Tip>
  苦手な食材が多すぎると、献立のバリエーションが限られる可能性があります。本当に避けたい食材のみを登録することをお勧めします。
</Tip>

### 食事制限

特定の食事スタイルや宗教的な理由による制限を設定できます。

**対応している食事制限:**
- **ベジタリアン**: 肉・魚を含まない
- **ビーガン**: 動物性食品を一切含まない
- **ペスカタリアン**: 魚は食べるが肉は食べない
- **ハラル**: イスラム教の食事規定に準拠
- **コーシャ**: ユダヤ教の食事規定に準拠
- **グルテンフリー**: 小麦などのグルテンを含まない
- **低糖質**: 炭水化物を制限

### 健康目標

個人の健康目標を設定することで、より適切な献立が提案されます。

<Tabs>
  <Tab title="ダイエット">
    **目標カロリー**: 1日の目標カロリーを設定
    
    例: 1,500 kcal/日
    
    体重減少を目指す場合、通常の摂取カロリーより500-1000 kcal少なく設定します。
  </Tab>
  
  <Tab title="筋肉増強">
    **タンパク質目標**: 体重1kgあたり1.6-2.2gのタンパク質
    
    例: 体重70kgの場合、112-154g/日
    
    筋肉の成長と回復に必要なタンパク質を確保します。
  </Tab>
  
  <Tab title="健康維持">
    **バランス重視**: 炭水化物50%、タンパク質20%、脂質30%
    
    バランスの取れた栄養摂取で、健康的な生活をサポートします。
  </Tab>
</Tabs>

## プロフィールの編集

既存のプロフィールを編集するには：

1. ダッシュボードの「Family Profile」に移動
2. 編集したいメンバーの「Edit」ボタンをクリック
3. 情報を更新
4. 「Save」をクリック

<Note>
  プロフィールを更新すると、次回の献立生成から新しい情報が反映されます。既存の献立は自動的には更新されません。
</Note>

## 複数プロフィールの管理

家族構成が変わる場合（例：子供の成長、家族の増減）に備えて、複数のプロフィールを作成できます。

**使用例:**
- 平日用プロフィール（両親と子供2人）
- 週末用プロフィール（祖父母も含む）
- 夏休み用プロフィール（子供の活動量増加を考慮）

## APIでのプロフィール管理

開発者の方は、APIを使用してプログラムでプロフィールを管理できます。

```javascript
// プロフィール作成
const response = await fetch('https://api.mealplan-ai.com/v1/profiles', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    family_members: [
      {
        name: '太郎',
        age: 35,
        gender: 'male',
        allergies: ['えび', 'かに']
      }
    ]
  })
});
```

詳細は[プロフィールAPI](/api-reference/profiles/create)をご覧ください。

## よくある質問

<AccordionGroup>
  <Accordion title="プロフィールは何人まで登録できますか？">
    無料プランでは4人まで、ベーシックプランでは6人まで、プレミアムプランでは10人まで登録できます。
  </Accordion>
  
  <Accordion title="子供の年齢が変わったら更新が必要ですか？">
    はい、年齢に応じて推奨カロリーや栄養素が変わるため、定期的な更新をお勧めします。誕生日に更新するのが良いでしょう。
  </Accordion>
  
  <Accordion title="アレルギー情報を間違えて登録した場合は？">
    すぐに編集して正しい情報に更新してください。安全のため、重要なアレルギー情報は慎重に確認することをお勧めします。
  </Accordion>
  
  <Accordion title="一時的なゲストの情報も登録すべきですか？">
    短期間のゲストの場合、献立生成時に「人数」を増やすだけで対応できます。長期滞在の場合は、プロフィールを作成することをお勧めします。
  </Accordion>
</AccordionGroup>
